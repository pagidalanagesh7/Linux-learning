## ✅ Day 3 — File Management & Linux Permissions

### File Operations

```bash
ls
cat
tac
cp
mv
head
tail
sort
grep
wc
```

### User & Group Commands

```bash
id
groups
```

### Linux Permissions

#### UMASK

```bash
umask
umask 022
```

#### CHMOD

```bash
chmod 777 file
chmod 644 file
chmod u+x file
```

#### CHOWN

```bash
chown user file
chown -R user directory
```

#### CHGRP

```bash
chgrp group file
```

### Utility Commands

```bash
file
diff
echo
```

### Interview Questions

* How UMASK works?
* Difference between 777 and ugo+rwx?
* Difference between chmod 400 and chmod ugo+r?
* Why user cannot open file even with write permission?
* Difference between `/root` and `/`?
